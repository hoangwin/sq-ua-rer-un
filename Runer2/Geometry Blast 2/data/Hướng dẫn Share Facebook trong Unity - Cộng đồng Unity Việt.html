Hướng dẫn Share Facebook trong Unity

17 September, 20143 October, 2014 Trương Toàn 1 Comment

Share Facebook trong game là một tính năng social rất hữu hiệu trong việc quảng bá game. Trong bài này mình sẽ hướng dẫn các bạn cách share Facebook trong Unity sử dụng Facebook SDK.

Bước 1: Trước tiên bạn cần có Facebook SDK cho unity – Download tại đây.

Bước 2: Tạo một App trên Facebook. Tạo App với link này. Sau đó ta có App ID của App vừa tạo. Với Android bạn phải tạo thêm Keyhash từ keystore để thêm thông tin cho App Trên Facebook.

 

Bước 3: Import Facebook SDK vào Unity sau đó chọn Facebook -> Edit Setting và điền App ID và App Name. Đối với Android bạn sẽ cần điền thêm mục PackageName và ClassName.

Bước 4: Đến đây mọi công việc chuẩn bị đã hoàn tất, việc còn lại là “Code” để post lên Facebook nữa thôi:)

– Ngoài ra bạn có thể tham khảo trên trang chính của Facebook SDK để thực hiện nhiều thao tác khác như : Post status, post link…

– Dưới đây mình tổng hợp lại các hàm để các bạn có cái nhìn tổng quát:

Core Methods

Name Description
FB.Init Initialize the SDK, required before doing anything else
FB.API Call the Facebook Graph API to find out about the user or act on his/her behalf
FB.Feed Trigger the Feed dialog for one-to-many sharing
FB.AppRequest Trigger the Request dialog for one-to-one sharing
FB.GetDeepLink Get the URL with which the app was invoked. Versions 4.3.3 and above.
FB.GameGroupCreate Prompt to create a new Game Group. Versions 6.0 and above.
FB.GameGroupJoin Prompt to join a Game Group. Versions 6.0 and above.

Auth Methods

Name Description
FB.Login Ask a user to authorize your app, or to grant additional authorization
FB.Logout Log a user entirely out of Facebook

 

Auxiliary Methods

Name Description
FB.Canvas.Pay Prompt the user to make a payment using Facebook Payments.Desktop only.
FB.PublishInstall Report an install, for integrating with Mobile Install Ads. Mobile only. DEPRECATED in v6.0
FB.ActivateApp Report an install, for integrating with Mobile Install Ads. Mobile only. Version 6.0 and later
FB.AppEvents.LogEvent Publish an app event, for deeper integration with Mobile App Install Ads. Mobile only, versions 4.3.3 and above.

Session Properties

Name Description
FB.UserId The Facebook user ID of the currently logged in user
FB.AccessToken The access token granted to your application by the current user
FB.IsLoggedIn Check whether a user is currently logged in and has authorized your app
